# JuiceSherpa

Juice Sherpa is a basic Rails that allows users to create juice recipes (yes, the juice you drink) and get live nutrition facts via the Nutritionix API.

## Live Demo

[JuiceSherpa](http://juicesherpa.com)

## Versions

- Ruby 2.2.0
- Rails 4.2
- PostgreSQL
- Heroku


## Installation

- `git clone`
- `bundle install`
- `rake db:migrate`
