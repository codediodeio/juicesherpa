# JuiceSherpa - A Rails App with Dynamic Nutrition Data

JuiceSherpa is a prototype app developed for CodeDiode.io. Step-by-step lessons show developers how to build the entire app from scratch.

## Versions

Ruby 2.2.0
Rails 4.2
PostgreSQL
Heroku


## Installation

`git clone`
`bundle install`
`rake db:migrate`
